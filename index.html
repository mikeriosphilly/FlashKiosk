Design rules
FontDM Sans only — weights 300/400/500/600. No display fonts in UI.
Logo placementAdmin screens only. Never shown in Kiosk Mode — hero is 100% event-focused.
Session type colorNeutral muted on all client screens. Color dots only visible in Setup list.
Hero image tintPurple→red gradient overlay at 18–22% opacity over any uploaded photo.
Primary CTABrand gradient (purple→red→orange) — one per screen maximum.
FABPurple #7B5CF5 — 48×48px, border-radius 14px. Event list only.
Dynamic Island90×29px black pill, 11px from top. Status bar height 52px total.
Tab bar18px bottom padding for home indicator. Active tab in purple, inactive in hint.
Checked In#4CAF50 green on #0C1F10 bg with #1A3A20 border.
Not Checked In#9B5070 rose on #1A1215 bg with #3A1E28 border.
Session typesMax 8. Preset palette, auto-assigned, user can reassign. No free color picker.
